Hackcovy Incubator Accelerator Program

by AngelHack & Kisstarup

After HackCovy Hackathon, 4 winning teams from both ideation track and acceleration track would be able to take part in HackCovy Incubation Program which is co-organized by AngelHack, United Nations Development Programme (UNDP), Hanoi Youth Union and an implementation partner: KisStartup. The HackCovy Incubation Program will officially start since June 15th 2020, last for 3 months, and aim at finished products that are ready for launching and calling for investment.
